# 🎶 Rythmix

Rythmix is a music streaming application that allow users to search for songs, like tracks and playlists, and
play music.



# ⚙️ Install
Clone the project.
```
git clone https://github.com/SuleAktas/spotify.git
```
Install required packages.
```
npm install
```
# 💻 Run
```
npm start
```
